Запрет правого клика на странице сайта

== Description ==

Плагин блокирует контекстное меню при нажатии правой кнопки мыши выводит предупреждение? а так же блокирует сочетание клавишь 'Ctrl + C', что позволит сохранить ваш контент.

Если вы хотите помочь в разработке этого плагина - добро пожаловать на [github](https://github.com/dyadyal/dl-block-right-mouse-button)

== Installation ==

1. Распакуйте архив с плагином
2. Загрузите папку с плагином в директорию /wp-content/plugins/ на вашем сервере
3. Активируйте плагин в разделе Плагины в панели администрирования WordPress
